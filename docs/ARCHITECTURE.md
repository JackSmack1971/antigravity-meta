# Antigravity Hybrid Architecture (2026)

## System Overview
The Antigravity Hybrid Agent Framework is structured as a closed-loop execution environment. All configuration, workflows, and logic are centralized within a universal `.agents/` directory, acting as a single source of truth.

## The Three Pillars
1. **Awesome-Skills Pack**: A repository of 1,400+ focused, atomic agent tools (e.g., framework scaffolds, database migrations).
2. **GSD (Get Shit Done)**: A deterministic, atomic execution pipeline enforcing rigid constraints (Spec → Plan → Execute → Verify → Commit).
3. **BMAD Method**: A simulated Agile software development squad layer (PM, Architect, QA, Dev) for multi-faceted reasoning and delivery.

## Directory Topology
- `.agents/skills/`: Unified manifest location for Awesome-Skills, GSD, and BMAD tooling.
- `.agents/rules/`: Enforced boundaries (Governance Layer).
- `.agents/bmad/`: BMAD framework configuration.
- `.agents/bmad-output/`: Immutable output artifacts generated by BMAD modules.
