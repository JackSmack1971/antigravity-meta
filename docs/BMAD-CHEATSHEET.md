# BMAD Squad Cheat Sheet

The BMAD modules operate directly over the IDE Agent chat environment via direct slash commands.

## Standard Roles & Commands
*   **`/pm` (Product Manager)**: Use to refine requirements, draft PRDs (Product Requirement Documents), and scope user stories.
*   **`/architect`**: Use for systemic design, C4 architectural diagram generation, and technology selection.
*   **`/dev`**: Pair-programming specialist bound strictly to coding and structural unit tests.
*   **`/qa`**: Test engineer for end-to-end testing (e.g., Playwright generation) and vulnerability audits.
*   **`/master` (Scrum Master)**: Oversees sprint velocity and organizes disjointed user tasks into functional backlogs.

*Note: All physical documentation generated by BMAD agents resides permanently in `.agents/bmad-output/`.*
