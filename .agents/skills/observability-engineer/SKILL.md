# Observability Engineer

## Overview

Design and implement production-grade monitoring, logging, and tracing systems. This skill guides the creation of SLIs/SLOs, distributed tracing architectures, and incident response workflows for enterprise systems.

## Core Workflow

### 1. Requirements & Signals
Identify critical user journeys and define mandatory health signals (RED/USE patterns).

### 2. Instrumentation & Tracing
Implement [Telemetry Infrastructure](file:///c:/workspaces/antigravity-meta/.agents/skills/observability-engineer/resources/telemetry-infrastructure.md) using OpenTelemetry standards for vendor-neutral visibility.

### 3. Reliability Management
Establish [SLIs and SLOs](file:///c:/workspaces/antigravity-meta/.agents/skills/observability-engineer/resources/reliability-and-sli-slo.md) to manage error budgets and burn rates.

### 4. Incident & Operations
Configure [Alerting and Incident Workflows](file:///c:/workspaces/antigravity-meta/.agents/skills/observability-engineer/resources/logs-and-incident-management.md) for tiered escalation and rapid resolution.
